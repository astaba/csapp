## Problem 11.7

I implemented the video capability with an over kill `player.js`
(generated by AI, my Javascript is lost rusty back in the days). The
`parse_uri` routine in the `tiny.c` file, requires a static string
`video_path` kept in a gitignored file in `csapp/.env.c.inc` for
privacy. Then in the end point video file the Javascript logic
requires a local `*.json` file listing all video for the gallery UI.
